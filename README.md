# Bean Tourney Package

Complete package for Bean Balatro Multiplayer Tournament.

## What's Included

This package contains everything needed for the Bean tournament:

- **Bean_BalatroMultiplayer** - Multiplayer mod with tournament features
- **HandyBalatro** - Quality of life improvements  
- **JokerDisplay** - Enhanced joker information display
- **libcrypto.dll** - OpenSSL 3.6.0 for signature verification

## Installation

Install via the Bean Balatro Multiplayer Launcher.

## Requirements

- Steamodded v1.0.0~BETA-1016c or later
- Lovely Injector (latest version)
